## Overview

This repository contains practice programs that cover the fundamentals of C programming, including:

- variables and basic I/O
- conditions and loops
- arrays, pointers, and functions
- file handling and dynamic memory
- recursion and structures
- pattern printing and mathematical logic

## Topics Covered

This repository includes exercises in the following areas:

- Basic syntax and simple calculations
- Decision-making with conditions
- Iteration and number logic
- Pattern generation
- 2D arrays and matrix-based problems
- Functions and pointers
- Dynamic memory allocation
- File reading and writing
- Preprocessor directives and macros

## Project Structure

The project contains practice programs under the following folders:

- basic/ - basic input/output, variables, and simple calculations
- conditionals/ - if/else, ternary operators, comparisons, and decision-making
- Loops/ - for, while, do-while, factorial, Fibonacci, prime checks, and number patterns
- patterns/ - star, number, hollow, and triangle pattern programs
- 2Darrays/ - 2D arrays, matrix operations, spiral/wave/transpose/rotation logic
- functions and pointers/ - functions, pointers, swapping, GCD, Pascal triangle
- Memory/ - dynamic memory allocation and array input/output examples
- File handling/ - file reading and writing examples
- preprocessor/ - macros, conditional compilation, and math-related examples
- recursion/ - recursive functions, factorial, Fibonacci, power, stair path, maze path, and tower of Hanoi
- structure/ - structures, unions, nested structures, typedef, and array of structures

## How to Run

Make sure you have a C compiler installed, such as GCC.

### Compile and run a program

```bash
gcc path/to/file.c -o program.exe
program.exe
```

### Example on Windows

```bash
gcc basic/helloworld.c -o helloworld.exe
helloworld.exe
```

### Quick Start

1. Open any folder such as `basic/`, `Loops/`, or `patterns/`
2. Choose a `.c` file
3. Compile it with GCC
4. Run the generated executable

## Learning Path

1. Start with `basic/`
2. Move through `conditionals/` and `Loops/`
3. Practice `patterns/` and `2Darrays/`
4. Learn `functions and pointers/`
5. Explore `Memory/`, `File handling/`, and `structure/`
6. Finish with `recursion/` and `preprocessor/`

## Notes

- Some folders also contain compiled `.exe` files generated during practice.
- These programs are meant for learning and experimentation, not for production use.
- You can compile any `.c` file individually using GCC.

This repository is a personal C programming practice collection that helps build a strong foundation in C language concepts through simple, focused examples.
